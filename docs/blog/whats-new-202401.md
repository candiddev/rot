## Enhancements

- Added `rot show-keys` to display all key names.
- Added `rot show-values` to display all value names.
- Changed `rot decrypt` to decrypt values from stdin by specifying `-`.

## Fixes

- Fixed `rot run` not passing stdin to commands.
